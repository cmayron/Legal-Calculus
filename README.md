🧪 Second‑Opinion Mode (8‑Step Pass)

Use this to sanity‑check any draft before filing.

1. State the goal (preserve right? cure defect? get time?).
2. Read the docket (what exists / what’s missing).
3. Pull FG micro‑modules.
4. Open paired LC/LCA modules.
5. Validate citations → TOA → email to yourself.
6. Attach a Proposed Order.
7. COS + filing plan (ECF/USPS + Mailbox Rule).
8. Archive to /packets/.


TL;DR:
Quickstart → FG checklist → LC module → save to /packets/.

---

🚨 Crisis Mode (Pro Se, Under Time Pressure)

1. Open FG Index → run the micro‑module matching your emergency
(e.g., FG‑47 TRO Steps, FG‑210 Service Checklist, FG‑800 Judicial Bias).
2. Pull matching template → fill → /s/ sign → export locked PDF.
3. Add a one‑line Proposed Order.
4. Validate citations + TOA → email to yourself.
5. File (ECF or USPS Certified + RRR).
6. If time: skim paired LC modules (Winter standard, Format is Force).
7. Archive in /packets/.


Crisis shortcuts:

• No Proposed Order → “nothing to rule” → strike.
• Wrong summons → service defect → dismissal risk.
• Formatting errors → clerk rejection.
• Mail delays → move for equivalent extension (Rule 6(b)).
• Repeat filings w/o stating what’s new → vexatious‑label risk.

🧼 Filing Hygiene (AI‑Era)

• Always include TOA; validate with a second system; email to yourself.
• Export locked PDF for court; keep .docx editable.
• Use Proposed Orders to avoid “nothing to rule” strikes.
• Document mail delays; move for extension when orders arrive late.
• Scrub PHI before committing; use /templates/redaction/.
• Keep PDFs < 25–35 MB; split exhibits if needed.
