Prompt Bank (copy into Copilot Pro)

• Explain — “Explain this block in plain steps and list preservation actions.”
• Map — “Create a doctrine map and sequence of filings for this issue.”
• MVF — “Rewrite as a Minimum‑Viable Filing: caption; one‑sentence ask; one paragraph facts; one paragraph reasons; signature; certificate of service.”
• Proposed Order — “Draft a one‑page proposed order granting the one‑sentence ask and setting any deadlines.”
• Red Team — “Attack this draft: list procedural objections and fixes in order of risk.”
• Metadata — “Produce a packet metadata header for commit with author, date/time, source, action.”

Practical Filing Checklist (copy into working note)

• Confirm jurisdictional e‑file rules and deadlines.
• Use correct caption, case number, and party names.
• Label exhibits; redact PII; include exhibit index.
• Confirm signature method and fee or waiver.
• Save EFSP receipt; screenshot; email packet to self; archive with metadata.


---

Archive Best Practices

• Commit final artifacts to /packets/ on desktop with the Packet Metadata Header.
• Save EFSP receipts, screenshots, and an emailed copy to yourself for timestamp.
• Encrypt packets when emailing; avoid public Wi‑Fi.


---

Troubleshooting (common fixes)

• Rejected for formatting → apply local formatting rules and refile.
• Missing signature or fee → add signature block and fee or waiver affidavit.
• Exhibit errors → label exhibits, create an index, ensure file size limits.
• Citation problems → add pin cites, jurisdiction, and negative treatment check.


---

Security Banner
Do not upload confidential, protected, or sensitive material to cloud AI. Redact PII before copy‑paste.
