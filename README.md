🧪 Second‑Opinion Mode (8‑Step Pass)

1. State the goal
2. Read the docket
3. Pull FG micro‑modules
4. Open paired LC/LCA modules
5. Validate cites → TOA → email yourself
6. Attach Proposed Order
7. COS + filing plan
8. Archive to /packets/


TL;DR: Quickstart → FG → LC → /packets/
